# Zenn販売ページ案

## タイトル

OpenAI Codex実践設計入門

## サブタイトル

AGENTS.md・Sandbox・CI・zapabob/codexで作るAI開発エージェント環境

## summary案

Codexを起動するだけで終わらせず、AGENTS.md、Sandbox、Approval、CI、zapabob/codexを通じて、AI開発エージェントが安全に働ける開発環境を作るための実践書です。

## 短い紹介文

この本は、Codexのコマンド集ではありません。  
Codexが安全に働けるリポジトリを作り、出力を検証し、日々の開発に組み込むための設計書です。

## 想定読者

- Codex CLIを触り始めたい開発者
- AIコーディングを日常開発に組み込みたい人
- AGENTS.mdを実務的に設計したい人
- Windows / WSL2環境でCodexを使いたい人
- 公式openai/codexとフォーク運用の考え方を学びたい人
